# Shell Development Rules (Add to CLAUDE.md)

## Modular Bash Workflow

This project uses a concatenation build model:
- **Source of truth**: `lib/XX-name.sh` fragments (edit these)
- **Assembled artifact**: generated by `just build` (do not edit directly)
- **Distribution**: the assembled script (unchanged behavior for end users)

### AI Development Rules

1. **Edit lib/ files, not the assembled script** — changes to the assembled
   script are overwritten by `just build`.

2. **Build and test after every change**:
   ```
   just build && just test
   ```

3. **One function per request** — write it, test it, commit it. Never write
   more than 50 lines without running `just lint`.

4. **Always provide `--help` output when asking AI to use an unfamiliar CLI tool.**

5. **Never guess shellcheck directives** — run `shellcheck` first to see the
   actual warning codes, then add targeted `# shellcheck disable=SCXXXX`
   comments only where necessary.

6. **Fragments can have split constructs** (if/fi across files) — this is
   expected and correct. The assembled script is the authoritative syntax check.
   Run `bash -n` only on the assembled output, not on individual fragments.

### Testing Strategy

| Test type | Command | What it checks |
|-----------|---------|----------------|
| Syntax | `just lint` | Shellcheck errors on assembled script |
| Format | `just fmt-check` | shfmt compliance |
| Build | `just build-check` | lib/ matches assembled artifact |
| Unit | `just test` | bats tests in test/*.bats |
| Integration | `just smoke` | Dry-run end-to-end |
| Full | `just check` | All of the above |

### Fragment Naming Convention

```
lib/00-header.sh      — shebang, set -euo pipefail, self-materialize
lib/01-common.sh      — colors, output helpers (ok/warn/fail/section)
lib/02-cli.sh         — arg parsing, usage(), interactive prompts
lib/03-*.sh           — early setup (run after CLI parsing)
lib/04-*.sh           — phase 1 installs
...
lib/17-finalize.sh    — summary, cleanup
```

Numbered 00-17; `just build` assembles in lexicographic order.
